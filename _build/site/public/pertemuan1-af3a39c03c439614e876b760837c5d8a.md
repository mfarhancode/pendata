## Pengertian Data Mining

**Data Mining** adalah proses menemukan pola, korelasi, anomali, dan informasi
tersembunyi yang berguna dari kumpulan data besar menggunakan teknik statistik,
matematika, kecerdasan buatan, dan machine learning.

> "Data mining is the process of discovering interesting patterns and knowledge
> from large amounts of data."

---

## Mengapa Data Mining Penting?

- Volume data terus bertumbuh sangat cepat (Big Data)
- Data mentah saja tidak cukup — perlu diekstraksi menjadi **pengetahuan**
- Digunakan di berbagai bidang: bisnis, kesehatan, keuangan, pendidikan, dll

---

## Tugas Utama Data Mining

| Tugas | Keterangan | Contoh |
|---|---|---|
| **Classification** | Mengelompokkan data ke kelas tertentu | Spam/Not Spam |
| **Clustering** | Mengelompokkan tanpa label kelas | Segmentasi pelanggan |
| **Association** | Menemukan hubungan antar item | Market basket analysis |
| **Regression** | Memprediksi nilai kontinu | Prediksi harga rumah |
| **Anomaly Detection** | Menemukan data tidak normal | Deteksi penipuan |

---

## Proses CRISP-DM

**CRISP-DM** (Cross Industry Standard Process for Data Mining) adalah proses standar
yang paling banyak digunakan dalam proyek data mining:

1. **Business Understanding** — Memahami tujuan dan kebutuhan bisnis
2. **Data Understanding** — Mengumpulkan dan mengeksplorasi data awal
3. **Data Preparation** — Membersihkan dan menyiapkan data untuk modeling
4. **Modeling** — Memilih dan menerapkan teknik modeling
5. **Evaluation** — Mengevaluasi model terhadap tujuan bisnis
6. **Deployment** — Menerapkan model ke lingkungan produksi
